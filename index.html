Micro-Task Registration and Tracking - Basketball Tournament and Game Fair

Event Information

Date: To be determined

Time: After lunch until 2:30 PM

Location: School gymnasium and adjacent spaces

Activities: Basketball tournament, game fair, food sales at the canteen

Entry: Tickets to be purchased (price to be determined)

This event is organized by students and includes a basketball tournament (including a teachers' team) as well as a game fair. B365 students will also run a canteen to sell food to players and spectators.

Information

Tasks Before the Event

Administration and Coordination

Description: Obtain all necessary administrative approvals for the event.

Micro-tasks:

  • Prepare authorization request documents for the event
  • Coordinate with administration and office staff
  • Obtain authorizations for the use of the gymnasium and spaces for the fair
  • Follow up on approvals
  • Communicate administrative decisions to the organizing team

Skills developed: Professional communication, administrative follow-up, organization

Sign up for this task

Tasks During the Event

Setup and Operations

Description: Prepare the venue before the start of the event (after lunch).

Micro-tasks:

  • Set up sports equipment in the gymnasium
  • Set up game fair booths
  • Install signage and posters
  • Prepare the canteen area
  • Verify that everything is ready before participants arrive

Skills developed: Logistics organization, teamwork, problem solving

Sign up for this task

Tasks After the Event

Cleanup and Reporting

Description: Clean and restore all spaces used during the event.

Micro-tasks:

  • Dismantle game fair booths and store materials
  • Clean the gymnasium floor
  • Return sports equipment to proper storage
  • Clean up the canteen area
  • Remove all signage and decorations
  • Ensure all spaces are returned to their original condition

Skills developed: Organization, teamwork, responsibility

Sign up for this task

Description: Compile and report on all financial aspects of the event.

Micro-tasks:

  • Collect and organize all receipts and financial records
  • Calculate total revenue from ticket sales
  • Calculate total revenue from canteen sales
  • Prepare a detailed expense report
  • Calculate the final profit/loss statement
  • Prepare a financial summary for administration

Skills developed: Financial management, accounting, reporting, attention to detail

Micro-tasks:

  • Create feedback forms for participants and spectators
  • Distribute feedback forms to team captains
  • Collect feedback from volunteers and organizers
  • Compile and organize all feedback data
  • Analyze feedback to identify strengths and areas for improvement
  • Prepare a summary report of feedback findings

Skills developed: Data collection, analysis, communication, critical thinking

Micro-tasks:

  • Create a list of all volunteers, staff, and contributors
  • Draft personalized thank you messages
  • Prepare certificates of appreciation
  • Organize a small recognition ceremony if appropriate
  • Send thank you emails to external partners and sponsors
  • Publish acknowledgments in school newsletter or website

Micro-tasks:

  • Compile tournament results and statistics
  • Document successes and challenges

  • Write executive summary for administration
  • Create recommendations for future events
